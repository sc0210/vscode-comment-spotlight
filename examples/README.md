# Examples

This folder contains example files demonstrating the Custom Comment Highlighter extension.

## Files

- **[test-example.js](test-example.js)** - JavaScript example showing various keyword highlights

## How to Use Examples

1. Install the extension (see main [README](../README.md))
2. Open any example file from this folder
3. Press `F5` if testing in development mode
4. See the keywords highlighted according to your settings

## Try It Out

Open `test-example.js` and you'll see:
- `// TODO:` highlighted in one color
- `// FIX:` highlighted in another color
- `// NOTE:` highlighted in a different color
- And more!

Toggle the highlighting on/off using the status bar button or Command Palette.
